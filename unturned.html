Game Server Login Tokens

  • Item Name: Game Server Login Tokens
  • Item Version: v1.0
  • Author: Steam Community
  • Support Forum: Steam API
  • License: None

Game Server Login Tokens (GSLTs) are used to log in Unturned dedicated servers to Steam, providing several benefits:

  • If using Server Codes to connect, your code will remain linked to your GSLT between sessions. Otherwise, you need to send your friends a new code each time you start the server.
  • Servers without a GSLT are considered “anonymous” and are hidden from the Internet server list.
  • Steam tracks your Favorites and History lists by address and port. With a GSLT, it can automatically transfer them if the server details change. This update should happen within approximately 24 hours.

Creating GSLTs

You can manually create GSLTs while logged in with your Steam account here: Create GSLTs. Use Unturned’s app ID 304930, and include a memo to remind you which server the token is for.

Unturned Configuration

The GSLT can be set in one of two places depending on your preference:

  • With the Login_Token property in each server’s Config.json file under the Browser section.
  • Using the GSLT command during startup. This can be specified in the Commands.dat file or on the command-line.

Automating GSLTs

Valve provides an IGameServersService web API for managing GSLTs. Consult their documentation here: IGameServersService API.
